Hi! I'm Joe Levinger.

I'm a full-stack web developer:

All JavaScript Ruby on Rails

Husband, Dad, Photographer, Hammered Dulcimer Player, Aspiring Web Developer...

After graduating from music school, I started my career as a software developer, programming in 3rd generation languages. Four years and two promotions later, though, I decided that it was time to pursue other interests. Then, in 2001, I jumped on the dot com bandwagon, went to web design school, and launched my own freelance web design business, which I have continued to this day.

As Ruby on Rails began to become more and more commonly used, though, it seemed only natural to learn a bit more about this relatively new technology. I started my Rails journey by completing an online Ruby on Rails bootcamp run by Tealeaf Academy, and branched out into JavaScript at General Assembly, in order to further hone my skills, ultimately combining my programming and web design backgrounds into a new career as a full-stack junior web developer.

While at Tealeaf, I created a Ruby on Rails blog as both a means of reinforcing the material I learned and as an aide memoire as well. Perhaps it will be of use to those of you who are beginning your Rails journey. Some of my posts:

  • Creating a Webhook and BDD
  • Beyond MVC, Creating Decorators and Service Objects, and Mocks & Stubs
  • Separating Actors, File Uploading, and Credit Card Processing
  • Concerns, Sidekiq, and Unicorn
